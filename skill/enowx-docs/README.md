# enowx-docs

A skill for exploring the **enowX** AI gateway's documentation. It teaches an
agent to discover enowX's live, machine-readable API catalog (`GET /api/docs`)
and explains each area — inference (`/v1`, `/anthropic`), provider accounts,
gateway keys, proxy pool, cloud sync, OTP numbers, and the Skill registry.

Install:

```bash
enx skill install enowx-docs        # into this project
enx skill install enowx-docs -g     # global
```
